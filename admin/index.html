How this admin works

Products are stored in Cloudflare D1, images are stored in Cloudflare R2, and the website reads the public product feed through the Cloudflare Pages Function API.

Deployment checklist

  1. Create D1 database and R2 bucket in Cloudflare.
  2. Add the bindings to this Pages project.
  3. Run the SQL migration and seed products.
  4. Create the first admin account from the Sign Up tab.
  5. Push the files to GitHub so Cloudflare redeploys.
